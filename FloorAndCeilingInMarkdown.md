# Using $\LaTeX{}$ floor function in markdown

So this is how you write floor(x) markdown:
$\lfloor x \rfloor$

`\lfloor` is for $\lfloor$.
`\rfloor` is for $\rfloor$.

# Using $\LaTeX{}$ ceiling function in markdown

This is how you write ceil(x) in markdown:
$\lceil x \rceil$

`\lceil` is for $\lceil$.
`\rceil` is for $\rceil$.
